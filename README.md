# Ticket-Flow-Analytics
Projeto acadêmico de análise de dados sobre Suporte de TI

# 📌 Descrição Breve do Projeto

Este projeto propõe a implantação de um modelo de gestão orientado por indicadores de desempenho para o setor de Suporte de TI, utilizando metodologia ágil baseada em Scrum e Power BI para análise de dados. A solução visa estruturar métricas como MTTR, SLA e produtividade técnica, promovendo melhoria contínua e tomada de decisão baseada em dados.
